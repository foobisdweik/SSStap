# Reference — Legacy SSTap 1.0.9.7 Artifacts

Artifacts from SSTap 1.0.9.7 used for reverse-engineering and UI/workflow parity.

| Folder / File | Purpose |
|---------------|---------|
| **Changelog.txt** | Version history, feature list |
| **readme.txt** | Original app description |
| **privoxy.conf** | Privoxy config (replaced by in-memory HTTP shim) |
| **lang/** | Localization files (.po, .mo) |
| **skins/** | UI skin bitmaps and res.ini |
| **tap-driver/** | TAP-Windows 9.x driver (replaced by Wintun) |

Not tracked: binaries (SSTap.exe, *.dll, etc.) — see `.gitignore`.
